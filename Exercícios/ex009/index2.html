Lista de Definições

Nas listas de definições usamos as tags:

  • dl = Definition List
  • dt = Definition Term
  • dd = Definition Description

As tags dt e dd possuem fechamento opcional.

Exemplo

HTML
Linguagem de marcação utilizada para criar o conteúdo de sites.
CSS
Linguagem de marcação para a especificação de estilos em sites
JavaScript
